## How to Download and Add an Image

1. Visit the Wikimedia Commons file page listed above.
2. Click **"Original file"** to get the exact `upload.wikimedia.org` URL with correct hash path.
3. Download to `content/img/<section>/filename.jpg` (or `.svg`).
4. Add a `{figure}` directive in the relevant `.md` file:

```markdown
```{figure} img/<section>/filename.jpg
:alt: <descriptive alt text>
:width: 70%

Caption text. Author Name, License, via Wikimedia Commons.
```
```

5. Commit with message prefix `images:` and update `CHANGELOG.md`.
